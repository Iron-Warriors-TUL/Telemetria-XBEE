# Telemetria-XBEE
Strona we flasku wyświetlająca parametry wysyłane (chwilowo mockowane) przez RPI z bolidu

---

## Środowisko wirtualne (opcjonalnie)

python -m venv venv  
venv\Scripts\activate      # Windows  
source venv/bin/activate   # Linux / macOS  

---

## Instalacja zależności i uruchomienie aplikacji

pip install -r requirements.txt  
cd backend  
python app.py  

Strona dostępna pod adresem domyślnym dla flaska:  
http://127.0.0.1:5000
